# Drop electric guitar samples here

Place 6 MP3 files named `E2.mp3 A2.mp3 D3.mp3 G3.mp3 B3.mp3 E4.mp3` in
this directory and the **Electric** instrument option will pick them up.

See `../README.md` for source and naming details. Until then, selecting
**Electric** in the instrument picker falls back to sawtooth synthesis.
